# Family Hub

Family Hub is a web-based family planner designed to help busy families organize daily activities, school events, and family responsibilities in one simple and intuitive place.

---

## Project Overview

Managing a family schedule often requires coordinating school events, extracurricular activities, and household tasks across multiple tools. Many existing solutions are either overly complex or not specifically designed for family use.

Family Hub addresses this problem by providing a lightweight, visually clear web application that allows users to manage family-related tasks efficiently without unnecessary features or setup.

---

## Project Goals

- Design a family-focused planning tool with a simple and intuitive user experience
- Apply clear visual hierarchy and information organization
- Build a responsive web application suitable for desktop, tablet, and mobile devices
- Use modern front-end development practices while keeping technical complexity appropriate for the project scope

---

## Target Users

- Busy parents and caregivers
- Families managing multiple schedules and activities
- Users looking for a lightweight alternative to complex calendar or productivity tools

---

## Core Features

- Add tasks with a title, date, and category
- Categorize tasks as School, Activities, or Family
- Display tasks in a clear, card-based layout
- Filter tasks by category
- Persist task data using browser local storage
- Responsive design across different screen sizes

---

## Technologies

- React  
- JavaScript  
- HTML  
- CSS  
- Browser LocalStorage  

This project is built using React to support a component-based architecture and improve state management.

---

## Project Status

This project is currently in active development.  
An initial proof of concept (POC) was created to validate core functionality, and this repository represents the full implementation of the final application.

The application will be deployed as a static web application and made publicly accessible via a live URL.
